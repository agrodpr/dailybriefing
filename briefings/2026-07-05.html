JadePuffer Ransomware Used AI Agent to Automate Entire Attack

Jul 4 · → Read more

Sysdig says JadePuffer is the first documented ransomware run entirely by an LLM agent — it broke in through CVE-2025-3248, an unauthenticated RCE in Langflow, then autonomously dumped databases, stole credentials, planted a cron-job backdoor, pivoted to production MySQL via Nacos, escalated privileges with CVE-2021-29441, and AES-encrypted 1,342 config items, adapting in real time when steps failed.
