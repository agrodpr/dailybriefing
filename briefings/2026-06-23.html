FFmpeg Fixes PixelSmash Flaw in Widely Used Video Decoder

June 22 · → Read more

PixelSmash (CVE-2026-8461, CVSS 8.8) is a heap out-of-bounds write in FFmpeg's MagicYUV decoder, triggered by a malicious AVI, MKV, or MOV file. Researchers achieved full remote code execution against Jellyfin 10.11.9 via its automatic library-scan pipeline, and the bug also causes denial-of-service in Kodi, Emby, Nextcloud, and OBS Studio.
